# Work Session Logging

After every meaningful unit of work — a feature completed, a bug fixed, a significant decision made — log a brief update to the Notion work log database:

**Database:** https://app.notion.com/p/da3b70463f674503bba639eec2616059?v=2ba952355bc04539942bd15789177851

Use the Notion MCP tools to create a new page/entry in that database with:
- What was done (1-2 sentences)
- Any decisions made or constraints discovered
- What's next

Log at natural stopping points, not after every file edit. The goal is for David to be able to open Notion and understand what happened in a session without reading the full transcript.
